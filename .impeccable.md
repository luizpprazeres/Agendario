# Agendario — Design Context

> **Brief completo:** `docs/DESIGN-BRIEF-AGENDARIO.md` (15 seções, todo o contexto)

## Design Context

### Users
**Luiz**, médico intensivista brasileiro plantonista (Recife). Workspace pessoal médico+financeiro. 80% mobile (entre plantões, em deslocamento), 20% desktop (fim de semana, fechamento mensal, declaração carnê-leão). Captura via Telegram durante o dia, volta na web pra **conferir, drilldown, relatórios**.

### Brand Personality
**Denso, polido, modular.** Três palavras: **competente**, **calmo**, **opinionado**. Workspace profissional pra dados financeiros pesados, com calma de premium consumer e flexibilidade de produtividade. NÃO é ferramenta de banco, NÃO é app fofo, NÃO é dashboard SaaS B2B.

### Aesthetic Direction
**Fusão Notion + Firefly III + Copilot Money:**
- **Densidade Firefly**: muita info por pixel, drill-downs, multi-account, rules visíveis
- **Polish Copilot**: animações sutis, tipografia caprichada, gráficos lindos, native iOS feel
- **Modularidade Notion**: blocos rearranjáveis, multiple views (table/calendar/kanban/gallery), command palette

**Theme:** dark default (`zinc-950`), light alternativo equally polished. Médico vê em ambiente clínico noturno e em casa de noite — dark é o contexto primário. Anti-references: glassmorphism, neon-on-dark, gradient text, "AI sass" cyan/purple.

### Design Principles
1. **Tabular nums em todo dado financeiro** — sempre `R$ 1.234,56` pt-BR
2. **Multiple views por entidade** — tabela default desktop, lista mobile, calendar/kanban/gallery opcionais
3. **Drill-down infinito** — toda métrica clicável vira drawer; nada é beco-sem-saída
4. **Mobile-first em interação, não em layout** — gestures (swipe, pull-refresh), FAB, bottom nav, mas conteúdo é o mesmo do desktop
5. **Captura natural via 1 input** — Quick Add (modal natural language) é parte do chrome global

### Anti-references (NÃO fazer)
- Glassmorphism, blur de fundo decorativo
- Border-left/right colorido em cards (banido absoluto)
- Gradient text
- Cards 3D, hover float dramático
- Recharts/Chart.js — tudo CSS/SVG inline
- Ícones decorativos (✨🎉💸) no chrome
- Mais de 2 famílias de fonte
- Confetes, toasts barulhentos

### Stack
React + Tailwind v4 + Lucide + Framer Motion + Radix primitives apenas. Para mockups standalone: Tailwind via CDN OK.
