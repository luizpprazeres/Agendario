4. **`amount_cents` é `decimal` (string)** no Drizzle — converter com `Number()` ou `String()` ao manipular
5. **Inngest hot-reload** funciona via PUT /api/inngest 200 (registra novas funções automaticamente — não precisa reiniciar)
6. **Bot precisa reiniciar?** Não — `tsx watch` cuida disso
